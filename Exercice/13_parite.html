Il utilise l'opérateur module %, qui calcule le reste de la division entière d'un nombre par un autre. Si le résultat du modulo d'un nombre par 2 est 0, alors ce nombre est pair. Voici quelque exemple d'utilisation de cet opérateur.

  • console.log(10 % 2); // Affiche 0 car 10 = 5 * 2 + 0
  • console.log(11 % 2); // Affiche 1 car 11 = 5 * 2 + 1
  • console.log(18 % 3); // Affiche 0 car 18 = 3 * 6 + 0
